Redux: Eliminating the Three-File Feature

Dec 06, 2016
javascript / react / redux

“Why do I have to touch three files to get a simple feature working? — Dan Abramov - You Might Not Need Redux This is a common complaint from those who are new to Redux. Redux provides a slew of benefits including, but not limited to, centralized state management and the ability to use pure functions for state changes. However, like all useful things, it adds indirection to your code. The Redux community sees this as a win and accepts the consequences, as do I, but I don’t believe that the infamous three-file feature is a necessary evil.
